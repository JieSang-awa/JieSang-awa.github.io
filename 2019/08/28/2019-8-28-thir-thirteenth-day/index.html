撰写于 2019-08-28 修改于 2019-09-29 分类 8月 标签 大杂烩

CSS样式实现方式

  1. 内联样式

    • 单独作用在 一个元素的上的 样式 叫 内联样式
    • 实现是通过 元素的 style 属性

  2. 内部样式

    • 针对单独的一个页面的 样式
    • 实现是 通过 style 标签实现的

  3. 外部样式

    • 针对 多个页面的 一般先定义成样式文件 然后在需要的页面上进行引入
    • 具体事项 先有样式 文件 ***.css
    • 使用 link 标签 或者 style 标签进行引入

  5. 后代选择器

  6. 子元素选择器

常用伪类

  • :link
    向未被访问的超链接添加样式

  • :visited
    向已被访问的超列检添加样式

  • :active
    向被激活的元素添加样式

  • :hover
    当鼠标悬停至元素上方时,向钙元素添加样式

  • :focus
    当元素获取焦点时,向该元素添加样式
